AI-powered system for analyzing human movement and detecting asymmetries using computer vision.  
**Tech:** Python, OpenCV, NumPy, Plotly, FastAPI  

---

### 🎥 [Real-Time Posture & Imbalance Detection](https://github.com/TataMutata/posture-detection)
Live video-based posture analysis tool for detecting asymmetry and movement deviations.  
**Tech:** Python, OpenCV, PyTorch

### 📊 [AI-Powered Analytics Dashboard](https://github.com/TataMutata/ai-analytics-dashboard)
